# 📚 John B. Watson | Behaviorismo

Este projeto é uma página web educativa desenvolvida em HTML, CSS e JavaScript que apresenta a vida e as contribuições de John B. Watson, fundador do Behaviorismo.

## 🧠 Sobre o Projeto

A aplicação tem como objetivo explicar de forma visual e interativa os principais conceitos do Behaviorismo, incluindo:

* Biografia de John B. Watson
* Surgimento do Behaviorismo
* Teoria do Estímulo → Resposta
* Experimento do Pequeno Albert
* Importância histórica
* Flashcards interativos para aprendizado

## 🚀 Funcionalidades

* 🎨 Interface moderna com design responsivo
* 📱 Layout adaptável para diferentes dispositivos
* 🧩 Cards informativos organizados em grid
* 🔄 Flashcards interativos (clique para virar)
* 🧭 Navegação com menu fixo (sticky)
* 🖼️ Imagens ilustrativas

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox + Grid + Animações)
* JavaScript (interatividade dos flashcards)
* Google Fonts (Poppins)
